
# 🔥 Gestion-de-cafeteria
El proyecto consiste en gestionar un sistema para registrar los tipos de café en una cafetería llamada **AromaCampus.**

## ✨ Estructura
El Sistema cuenta cosn la siguientes funcionalidades

- ✅ Main
- ✅ **feature/add-coffe**: Permite agregar La infromacion sobre los cafés (Nombre, descripcion, precio y disponibilidad)
- ✅ **feature/edit-coffe**: Permite editar la infromación.
- ✅ **feature/delete-coffe**: Permite eliminar infromación,
- ✅ producto.md (Archivos guardados)

# 📌 Funciones
Cada Funcionalidad cuenta con un archivo .py que permite la correcta ejeción.
-agregar_cafe.py
-eliminar.py
-editar.py
